# Mise en Place — Recipe & Meal Planner
*Goal: Mobile-first app to track home recipes, plan weekly meals, and generate smart grocery lists.*

## What's built

Next.js 16 + Supabase + Claude AI, deployed to Vercel. PWA-installable on Android.

- [x] Recipe library — add by name, AI auto-fills ingredients, cuisine, cook time
- [x] Cooking history — log sessions with ratings and notes
- [x] Weekly meal planner — assign recipes to days, navigate weeks
- [x] Grocery list — auto-generated from weekly plan, grouped by category, shareable
- [x] AI recommendations — 5 new recipe ideas based on cooking history
- [x] Magic link auth — no password, sign in via email link
- [x] PWA — installable via Chrome "Add to Home Screen"
